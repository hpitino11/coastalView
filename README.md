# 🌊 Coastal View Insurance Website

A fully responsive, production-ready insurance agency website built for a real client. This project showcases full-stack development, modern UI design, and deployment to a live environment.

🔗 **Live Site:** https://coastalviewins.com  
💻 **Repository:** https://github.com/hpitino11/coastalView  

---

## ✨ Overview

This project was designed and developed from scratch for a local insurance agency, with a focus on clean UI, performance, and real-world usability.

The site provides detailed service pages (Auto, Homeowners, Business, Flood, Watercraft, Builders Risk), along with a fully functional contact system and secure deployment.

---

## 🚀 Features

- Responsive, mobile-friendly design
- Custom service pages for multiple insurance types
- Integrated contact form with backend API
- Spam protection (rate limiting + honeypot field)
- Secure HTTPS deployment with DNS & SSL configuration
- Optimized assets for performance and fast load times

---

## 🛠️ Tech Stack

**Frontend**
- HTML5, CSS3, JavaScript
- Bootstrap (customized styling)

**Backend**
- Node.js
- Express.js
- Resend / Nodemailer (email handling)

**Deployment & Tools**
- GoDaddy (hosting + DNS management)
- SSL/TLS configuration
- Git & GitHub for version control

---

## 🔒 Contact API (Backend)

The contact form is powered by a custom Express API with:

- Input validation
- Rate limiting (prevents spam/abuse)
- Honeypot field for bot detection
- Secure environment variable handling
